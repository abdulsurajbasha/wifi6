### Project Name: WIFI6
### Industry Type :Edutech
### Problem Statement: " With the increasing demand for higher data speeds, improved capacity, and enhanced connectivity experiences, the emergence of WiFi 6 (802.11ax) technology has become a pivotal advancement in wireless networking. This project aims to implement and optimize WiFi 6 in a real-world scenario, exploring its capabilities to deliver superior performance in diverse environments with the help of azure technologies like azure bot and wifi6 explianation video from blob storage "
